HTML

HTML stand for Hyper Text Markup Language
Created by Tim Berners-lee in 1993
Specifically created to make World Wide Web pages
File Extension ".htm" or ".html" .

Features:

    Non-case sensitive language
    Tags are surrounded by two characters "<" and ">" (angular backets )
    Html tags normally come in pairs line and
    First tag in pair is start tag/opening tag and the second tag is end tag/closing tag.
    Text between the start tag and the close tag is element content.
